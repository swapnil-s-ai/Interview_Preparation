# Practice Strategy

For every topic:

1. Solve without AI.
2. Optimize the solution.
3. Explain the approach aloud.
4. Compare with Pythonic implementation.
5. Record mistakes.

# Practice Rules

- Prefer clean code over clever code.
- Always discuss time and space complexity.
- Write meaningful variable names.
- Think aloud while solving interview questions.
- Learn at least one Pythonic solution for every problem.
